Tsukuba and Segregation

Alan Engel
Last update: 26 November 2020

Objective: Apply Elbers decomposition method to segregation in and around Tsukuba.

A recently published decomposition method (Elbers, 2021) enables changes in segregation to be decomposed into changes resulting from new units (for example, occupations), disappearing units, changes in groups (for example, gender), changes in units, and structural changes. Elbers has released an R package with clearly written vignette and documentation. This project will apply this method to Tsukuba and its surroundings using readily available census and other government statistics.
