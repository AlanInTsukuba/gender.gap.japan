Gender Gap Japan

Alan Engel
Last update: 16 November 2021

Objective: Apply segregation statistics to government data around and about Tsukuba.

A recently published decomposition method (Elbers, 2021) enables changes in segregation to be decomposed into changes resulting from new units (for example, occupations), disappearing units, changes in groups (for example, gender), changes in units, and structural changes. Elbers has released an R package with clearly written vignette and documentation. This project will apply this and other segregation statistics methods to gender gaps in Japan using readily available census and other government statistics.

    1. Basic walkthrough of Elbers's segregation package using Japanese population census data

      Elbers's R package "segregation" contains a detailed vignette (https://elbersb.github.io/segregation/articles/segregation.html) that steps through the main features of his package. This basic walkthrough uses data on occupation versus gender from the 1995 and 2000 Japan Population Censuses.

  2. Reproduction of Uchikoshi and Mugiyama (2020) with extension

    Brinton and Ngo (1993) published a landmark study of gender and occupation in Japan. Uchikoshi and Mugiyama (2020) look at subsequent trends at a more detailed minor occupation level. The latter constructed a occupational crosswalk that covers Japanese population censuses that were conducted between 1980 and 2005, and rely mainly on the Duncan dissimilarity index as their measure of segregation. The objective of this subproject is to partially reproduce the Uchikoshi and Mugiyama (2020) results, then compare them to results using Elbers's decomposition.

    1. Getting the data for harmonizing 1985-2005 occupation classifications

      A step by step guide to downloading occupation and gender data from Japanese population censuses.

    2. Preprocessing the data

      Data downloaded from e-Stat usually must be processed before it can be incorporated into statistics packages or, in this case, used for constructing an occupational crosswalk. Instead of the 2005 data downloaded in the previous step, here I use 2000 data because it has a "feature" that is not in the 2005 data and must be corrected.

    3. Harmonizing 1985-2005 occupation classifications

      The objective is to create occupational crosswalk covering population censuses between 1985 and 2005. I omit the 1980 census because it is available only in PDF image format.

    4. Build master dataset

      The occupational crosswalk used to build a master dataset from the preprocessed occupation-gender data.

    5. Descriptive statistics

      This task produces a table of descriptive statistics similar to Table 3 in Elbers (2021). Except where noted, the statistics are generated using the R scripts included.

    6. Uchikoshi/Mugiyama decomposition R development

      Develop R functions and script for computing Uchikoshi/Mugiyama decomposition.

    7. Reproduce results from Uchiyama/Mugiyama

      Use R to reproduce results.
